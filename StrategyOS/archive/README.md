# Archive

Processed materials from `/inbox` are stored here.

---

## Structure

Each processing cycle creates a dated folder:

```
archive/
└── YYYY-MM-DD-descriptive-slug/
    ├── processing-notes.md    ← The implementation plan that was approved
    └── {original files}        ← Source materials from inbox
```

---

## Contents

- **processing-notes.md** — Documents what changes were made and why
- **Source materials** — Original files, preserved as received

---

## Usage

- Reference archived materials from strategy files: `(Ref: /archive/2025-01-15-competitor-analysis/)`
- Archives are read-only records—don't modify after creation
- Use for audit trail and understanding strategic evolution
